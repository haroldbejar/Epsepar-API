# Propuesta de Modernización

El objetivo es migrar la solución a una arquitectura moderna basada en Clean Architecture y .NET 8, integrando buenas prácticas de desarrollo, automatización y despliegue continuo.

## Lineamientos

- Separación de capas (Domain, Application, Infrastructure, WebApi)
- Uso de EF Core y MySQL
- Preparación para CI/CD
- Documentación y comunicación técnica continua
