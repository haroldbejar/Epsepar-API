# Documentación Técnica Inicial

## Descripción General

Este proyecto implementa Clean Architecture sobre .NET 8, separando responsabilidades en cuatro capas principales:

- **Domain**: Modelos y lógica de negocio.
- **Application**: Casos de uso y lógica de aplicación.
- **Infrastructure**: Acceso a datos, servicios externos, implementaciones técnicas.
- **WebApi**: Exposición de endpoints HTTP.

## Dependencias Iniciales

- .NET 8 SDK
- Entity Framework Core
- MySQL
